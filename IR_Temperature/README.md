#IR Temperature from SensorTag

SensorTag has IR Temerature sensor, can measure both object and ambient temperature.


__Sensor Overview__


| Name   | Update  | Size per Update | I2C Address | I2C Speed |
|--------|---------|------------|--------|---------|
| TMP006 | > 250 ms |  2 x 16bit |	0x44 |  < 3.4 MHz |
